BORDER

[Statement] - changes screen border colour

Syntax:

BORDER color

Parameters:

color: ZX Spectrum Colors 0-7

Description:

This can be used to change the border colour. There is no possibility to use BRIGHT.

Color values are:

0 - black
1 - blue
2 - red
3 - purple, technically called magenta
4 - green
5 - pale blue, technically called cyan
6 - yellow
7 - white

Examples:

REM 'Change Border to black
LET col = 1
BORDER col
REM 'now border is blue

Notes:

  • This function is 100% Sinclair BASIC Compatible

See also:
