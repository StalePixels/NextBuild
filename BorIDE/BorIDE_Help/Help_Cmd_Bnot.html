bNOT

[Function] - Binary NOT of one value

Syntax:

bNOT a

Parameters:

a: One Flag

Description:

Performs the Logical Negation and returns TRUE if the arguments is False and vice versa.

Examples:


PRINT bNOT 1

The first of above function, when called, returns 0.

NOT a
a Result
False True
True False

Notes:

  • This function does not exist in Sinclair BASIC.
  • It can be also written as "~", eg: ~1

See also:

bAND, bOR, bXOR

Supported Plattforms:

ZX Spectrum
